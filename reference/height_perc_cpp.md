# Calculate dominant diameters and heights for simulations for angle-count plots.

Calculate dominant diameters and heights for simulations for angle-count
plots.

## Usage

``` r
height_perc_cpp(rho_seq, z, rho)
```

## Arguments

- rho_seq:

  BAF sequence for simulations

- z:

  vector with heights of trees

- rho:

  number of trees considered for calculating diameter and dominant
  height

## Value

Data frame.
