# Calculate weighted geometric mean.

Calculate weighted geometric mean.

## Usage

``` r
weighted_mean_geom(x, w)
```

## Arguments

- x:

  attribute for which calculate mean.

- w:

  vector with weights associated to x.

## Value

Data frame.
